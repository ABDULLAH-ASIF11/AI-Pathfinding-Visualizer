# 📖 Overview

The **AI Pathfinding Visualizer** is an educational desktop application that demonstrates how different pathfinding algorithms explore a grid to discover the shortest route between a **Start Node** and a **Goal Node**. 

The project provides an intuitive graphical interface where users can build custom environments, generate random mazes, compare multiple search algorithms, and analyze their performance using real-time metrics. This project was developed as part of an **Artificial Intelligence** assignment to demonstrate practical implementation of classical search algorithms.

---

# ✨ Features

### Interactive Grid
* Dynamic Grid Resizing | Interactive Cell Editing
* Start/Goal Node Selection | Wall Placement/Removal
* Random Maze Generation with Adjustable Density

### Search Algorithms
* **Breadth First Search (BFS):** Complete, Optimal (for unit costs), Uninformed.
* **Uniform Cost Search (UCS):** Complete, Optimal, Cost-Based expansion.
* **Greedy Best First Search (GBFS):** Informed, Heuristic-driven, Fast exploration.
* **A* Search:** Informed, Optimal, Efficient, balances cost + heuristic.

### Real-Time Visualization
* **Frontier:** 🟨 Yellow | **Expanded:** 🟦 Blue | **Final Path:** 🟩 Green
* **Nodes:** 🟧 Start | 🟥 Goal | ⬛ Obstacles
